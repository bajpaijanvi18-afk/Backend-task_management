# Full Stack Task Management Application

A production-style full-stack task management system built using:

- React.js
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Authentication

---

# Features

## Authentication
- User Registration
- User Login
- JWT Token Authentication
- Password Hashing using bcrypt
- Protected Frontend Routes
- Protected Backend APIs

---

## Task Management
- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- User-specific task isolation

---

## Admin Features
- Role-Based Access Control
- Admin-only protected routes

---

# Tech Stack

## Frontend
- React.js
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT
- bcryptjs
